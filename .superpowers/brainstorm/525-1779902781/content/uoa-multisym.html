Switching tabs loads that symbol's tape · All symbols share the same filters (window, score, intent, structure) · Each tab shows a live signal count badge and net premium direction
Key elements in this design:
WATCH row
Symbol chips below the top bar. Click to make active. ✕ to remove. Type to add new ticker.
Tabs
One tab per symbol. Each shows live signal count + net bull/bear premium at a glance.
Shared filters
Window, minScore, intent, structure, 0DTE apply to ALL symbols simultaneously.
Dot indicator
Tab dot color = net bias (green=bull dominant, red=bear dominant, amber=mixed).
