# Sketch Manifest

## Design Direction
Institutional-grade dark terminal tape — Bloomberg TOMS / Flow X aesthetic. Monospace numerics, thin borders, no chrome. Black background (`#0a0a0f`), surface layers in near-black. Color used only for signal: green bull, red bear, amber mid-score, blue high-score, green elite-score. The score chip + intent arrow is the primary attention anchor on each row.

## Reference Points
- Bloomberg TOMS tape
- Flow X options flow scanner
- Existing GEX app: same dark Tailwind palette, `--surface-1`, `--text-1/2/3`, monospace numerics
